Quick Start

Getting Started with Image Streaming:
  1. Configure your image sources in config/streams.json
  2. Start the application and visit the Stream Status page
  3. Access your RTSP streams at rtsp://localhost:8554/{stream_name}
Getting Started with Roof Status Monitor:
  1. Visit the Roof Status page to view the dashboard
  2. Submit status updates via POST to /tools/roof-status/api/sfro-roof-status
  3. Monitor building status updates in real-time
